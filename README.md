# BankAccount — Учебный проект

Реализация банковского счёта с методами депозита, снятия и отслеживания истории операций.

## Функционал

- Депозит (`deposit`) — внести деньги на счёт
- Снятие (`withdraw`) — снять деньги (с валидацией)
- Баланс (`getBalance`) — получить текущий баланс
- История (`getTransactionHistory`) — последние 5 операций
- Валидация — защита от отрицательных сумм и перерасхода
